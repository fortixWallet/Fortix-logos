# FortiX Crypto Logos

Official logo assets for FortiX Wallet.

## Structure

```
/chains     - Chain/network logos (114 logos)
/native     - Native token logos by symbol (140 logos)
/tokens     - Token logos by chainId/address (15,346 logos)
```

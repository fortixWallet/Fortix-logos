# FortiX Crypto Logos

Official logo assets for FortiX Wallet.

## Structure

```
/chains     - Chain/network logos (111 logos)
/native     - Native token logos by symbol (144 logos)
/tokens     - Token logos by chainId/address (14,330 logos)
```

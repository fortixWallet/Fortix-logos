# FortiX Crypto Logos

Official logo assets for FortiX Wallet.

## Structure

```
/native     - Native token logos (ETH, BTC, SOL, etc.)
/chains     - Chain logos by chainId (future)
/tokens     - Token logos by chainId/address (future)
```

## Usage

```
https://raw.githubusercontent.com/user/fortix-logos/main/native/eth.png
```
